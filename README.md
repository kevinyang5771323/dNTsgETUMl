# 健身房管理系统 java513

## 项目概述


## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 运行环境
- 系统环境：Windows/Mac

### 开发环境
- 开发工具：IDEA
- JDK版本：1.8
- 项目构建：Maven
- 数据库：Mysql

### 技术框架
- 后端：Java、SpringBoot、Mybatis
- 前端：Vue

## 功能模块

### 系统角色
- 管理员
- 会员

### 功能列表
- 用户登录
  - 管理员登录
  - 会员登录
- 管理员功能
  - 会员管理
  - 健身房管理（包含课程管理，器材管理）
  - 员工管理
  - 报名表管理
- 会员功能
  - 课程查询
  - 在线充值
  - 充值记录查询
  - Vip管理
  - 购买记录查询
  - 我的资料管理
  - 修改密码

## 部署步骤

1. 准备开发环境：安装IDEA、JDK1.8、Maven、Mysql。
2. 克隆或下载项目源码。
3. 导入项目到开发工具中。
4. 配置数据库连接信息。
5. 执行数据库脚本，初始化数据。
6. 编译并运行后端服务。
7. 前端项目构建并运行。
8. 访问前端项目，进行系统登录和操作。
